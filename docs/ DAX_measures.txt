1Ô∏è‚É£ HEADCOUNT MEASURES
==============================================

-- Total number of employees in dataset
Total Employees = COUNTROWS('HR_Employee_dataset')

-- Employees currently with the company (Attrition = "No")
Active Employees = 
CALCULATE(
    COUNTROWS('HR_Employee_dataset'),
    'HR_Employee_dataset'[Attrition] = "No"
)

-- Employees who have left the company (Attrition = "Yes")
Terminated Employees = 
CALCULATE(
    COUNTROWS('HR_Employee_dataset'),
    'HR_Employee_dataset'[Attrition] = "Yes"
)

-- Employee Count (alternative name)
Employee Count = COUNTROWS('HR_Employee_dataset')

-- Distinct Employee Count (safety check - should match Total Employees)
Distinct Employees = DISTINCTCOUNT('HR_Employee_dataset'[EmployeeNumber])

2Ô∏è‚É£ ATTRITION MEASURES
==============================================

-- Basic attrition rate (terminations / total employees)
Attrition Rate = 
DIVIDE(
    [Terminated Employees],
    [Total Employees],
    0
)

-- Formatted as percentage
Attrition Rate % = 
FORMAT([Attrition Rate], "0.0%")

-- Attrition count (same as Terminated Employees)
Attrition Count = 
CALCULATE(
    COUNTROWS('HR_Employee_dataset'),
    'HR_Employee_dataset'[Attrition] = "Yes"
)

-- Attrition rate by department (dynamic)
Attrition % by Dept = 
DIVIDE(
    [Attrition Count],
    CALCULATE(
        COUNTROWS('HR_Employee_dataset'),
        ALLEXCEPT('HR_Employee_dataset', 'HR_Employee_dataset'[Department])
    ),
    0
)

-- Active employee percentage
Active % = 
DIVIDE(
    [Active Employees],
    [Total Employees],
    0
)

üìù NOTES FOR USAGE
==============================================

1. All measures assume table name is 'HR_Employee_dataset'
   - If your table name differs, replace in all formulas

2. Calculated columns (section 8) need to be created separately
   - Go to Modeling ‚Üí New Column
   - Uncomment and paste each column formula

3. For KPI visuals, use:
   - Indicator: [Attrition Rate %]
   - Trend axis: HireDate (by Month)

4. For Gauge visuals, use:
   - Value: [Avg Performance Rating]
   - Minimum: [Min Performance]
   - Maximum: [Max Performance]
   - Target: [Target Performance] (optional)

5. Format percentages as "0.0%" in Visualizations pane

6. Format currency as "$#,##0"
